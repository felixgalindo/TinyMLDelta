# Security Policy

TinyMLDelta is security-sensitive in that it touches model updates and
firmware-level behavior.

## Reporting a Vulnerability

If you discover a security issue related to TinyMLDelta:

1. **Do not** open a public GitHub issue.
2. Instead, contact the maintainer privately (e.g., via email or a direct channel).
3. Provide enough detail to reproduce the issue.

We will acknowledge your report and work on a fix. Once a patch is available,
we will coordinate an appropriate disclosure.
